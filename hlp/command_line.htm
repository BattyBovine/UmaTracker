Command line

UmaTracker may be invoked from the command line to open a file, or to export an UTM to another file type.

Open

UmaTracker [UTM filename]

This command line will start UmaTracker and open the specified UTM document.

Export

UmaTracker [UTM filename] -export [export file] [log file] [DPCM file]

This command line will export the specified UTM file to the specified export file. Any messages created by the export process will be sent to the specified log file. After exporting, UmaTracker will close without opening its main window. This is intended for automated processing of UTM files.

The type of export is determined by the file extension used. Known export file extensions: NSF, NES, BIN, PRG, ASM, TXT. If the export file extension is not known, the first custom export plugin will be tried instead, if available. The DPCM file is only required for BIN export; it is ignored in other cases.

Disable crash dumps

UmaTracker [UTM filename] -nodump

This command will disable the crash dump generation, for cases where these are undesirable.
