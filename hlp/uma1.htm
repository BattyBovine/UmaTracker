BattyBovine UMA1

About

The UMA1 contains two pulse wave channels that are identical to the internal pulse channels. It has yet to be used in a game, but hopefully will be someday.

Check 2A03 for more details about the square channels.

Instruments

The instruments used by MMC5 are similar to the 2A03. The main difference is that there are now eight duty cycle options: 6.25%, 12.5%, 18.75%, 25%, 37.5%, 50%, 62.5%, and 75%. See 2A03 instruments for more information.
